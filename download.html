System Requirements

The InVEST tool suite consists of open source Windows based binaries except for the coastal protection model that requires ArcGIS. InVEST takes GIS data as input and output, so a desktop based GIS tool is required. We recommend QGIS


The Coastal Protection model has additional software requirements

  • ArcGIS 9.3 (service pack 1 or higher) or ArcGIS 10 (service pack 1 or higher), ArcGIS 10.1, or ArcGIS 10.2.
  • ArcGIS Spatial Analyst extension
  • Marine Python Extensions for ArcGIS 9.3/Python 2.5" Required for the ArcGIS based coastal protection model.
  • Marine Python Extensions for ArcGIS 10/Python 2.6 Required for the ArcGIS based coastal protection model.
  • Marine Python Extensions for ArcGIS 10.1 and 10.2/Python 2.7 Required for the ArcGIS based coastal protection model.
