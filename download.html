InVEST 2.2.1 was released on January 26, 2012. It includes bug fixes to the Coastal Protection tool, Arc 9.3 support to the InVEST ArcMap document, includes the full InVEST 3.0 Beta suite on the "Models Only" installer, as well as minor typo fixes in the user's guide.

System Requirements

  • ArcGIS 9.3 (service pack 1 or higher) or ArcGIS 10 (service pack 1 or higher)
  • ArcGIS Spatial Analyst extension

User Support

A user's guide is included with InVEST and is also available online here: Online InVEST 2.2.1 User's Guide

Users are encouraged to submit bug reports directly to our issue tracker. Reports can also be directly emailed to richsharp@stanford.edu. When submitting a bug report please include the following information:

  • InVEST model you're having trouble with.
  • Explicit error message or behavior (a copy and paste of the log output is helpful).
  • If possible, a screenshot of the state of your InVEST toolset when you get the error.
  • ArcGIS version and service pack number.

Supporting documents including old documentation, data source references, and InVEST presentations are temporarily hosted at invest.ecoinformatics.org/tool-documentation.

Supporting Downloads

These downloads are required for some models. Refer to the user's guide for a particular model for more information.

  • Geospatial Data Abstraction Library (GDAL) Extension Only required for pollination model.
  • Run "Marine Python Extension Check" in the InVEST 2.1 toolbox to see if you need to install one of these:
    • Marine Python Extensions for ArcGIS 9.3/Python 2.5
    • Marine Python Extensions for ArcGIS 10/Python 2.6
  • Modified Hargreaves Referece Evapotranspiration Tool (ArcGIS 9.2)
  • Excel tool to estimate Extraterrestrial Radiation
