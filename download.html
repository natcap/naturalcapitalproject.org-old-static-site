The ArcGIS based tools have additional software requirements

  • ArcGIS 9.3 (service pack 1 or higher) or ArcGIS 10 (service pack 1 or higher)
  • ArcGIS Spatial Analyst extension
  • Marine Python Extensions for ArcGIS 9.3/Python 2.5 Only required for ArcGIS marine models.
  • Marine Python Extensions for ArcGIS 10/Python 2.6 Only required for ArcGIS marine models.
  • Marine Python Extensions for ArcGIS 10.1/Python 2.7 Only required for ArcGIS marine models.
