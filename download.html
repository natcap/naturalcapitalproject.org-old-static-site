InVEST 2.4.3 was released on October 15, 2012.

InVEST 2.4.0 is a major release which releases two additional beta versions of the InVEST models in the InVEST 3.0 framework. InVEST 2.4.1 was released shortly after InVEST 2.4.0 to fix some compatibility issues relating to ArcGIS 9.3. InVEST 2.4.2 was released to fix a path issue with the included ArcMap document and includes some more memory-efficient processing for Biodiversity 3.0 Beta. InVEST 2.4.3 was released to fix several minor bugs in Carbon Biophysical 3.0 Beta, Hydropower 3.0 Beta, and adds some new features to the InVEST user interface and installer. Additionally, this release introduces start menu shortcuts for all available InVEST 3.0 beta models. Existing InVEST 2.x models can still be found in the included Arc toolbox.

Existing InVEST models migrated to the 3.0 framework in this release include:

  • Biodiversity 3.0 beta
    • Minor bug fixes and usability enhancements
  • Overlap Analysis 3.0 Beta
    • Minor bug fixes and usability enhancements.
    • Split into two separate tools to enable the use of management zones.

Updates to existing ArcGIS models:

  • Coastal vulnerability
  • Coastal protection (erosion protection)

Additionally there are a handful of other minor fixes and feature enhancements since the previous release:

  • Biodiversity 3.0 Beta now uses is now significantly more memory-efficient for much of its processing. This will affect users with very large input rasters.
  • InVEST 3.0 Beta standalones can now be launched from the Windows Start Menu (Start Menu -> All Programs -> InVEST 2.4.3)
  • Big fixes for the calculation of raster statistics
  • InVEST 3.0 Wave Energy no longer requires an AOI for global runs, but it does encounter memory issues on machines with less than 4GB of RAM. This will be fixed in a minor release.
  • Minor fixes to several chapters in the user's guide
  • Harvest maps are no longer required inputs to the 3.0 Beta Carbon model
  • Improved installer allows user to remove InVEST from Windows Add/Remove Programs
  • Other minor bug fixes and runtime performance tweaks in the 3.0 framework

System Requirements

  • ArcGIS 9.3 (service pack 1 or higher) or ArcGIS 10 (service pack 1 or higher)
  • ArcGIS Spatial Analyst extension

User Support

A user's guide is included with InVEST and is also available online here: Online InVEST 2.4.3 User's Guide

Users are encouraged to submit bug reports directly to our issue tracker. Reports can also be directly emailed to richsharp@stanford.edu. When submitting a bug report please include the following information:

  • InVEST model you're having trouble with.
  • Explicit error message or behavior (a copy and paste of the log output is helpful).
  • If possible, a screenshot of the state of your InVEST toolset when you get the error.
  • ArcGIS version and service pack number.

Supporting documents including old documentation, data source references, and InVEST presentations are temporarily hosted at invest.ecoinformatics.org/tool-documentation.
