InVEST 2.1 beta (old version)

Supporting documents including old documentation, data source references, and InVEST presentations are temporarily hosted at invest.ecoinformatics.org/tool-documentation as we transition to an interactive and more functional website.

A user support forum (registration required) is hosted at invest.ecoinformatics.org/discuss. The forum will migrate directly to this site in the coming months.

System Requirements

  • ArcGIS 9.3 (service pack 1 or 2) or ArcGIS 10 (service pack 1)
  • ArcGIS Spatial Analyst extension
  • Python
    • Version 2.5 or higher required for ArcGIS 9.3
    • Version 2.6 or higher required for ArcGIS 10
  • Other tools as necessary
    • Geospatial Data Abstraction Library (GDAL) Extension
    • Marine Python Extensions for ArcGIS 9.3/Python 2.5
    • Marine Python Extensions for ArcGIS 10/Python 2.6
    • Modified Hargreaves Referece Evapotranspiration Tool (ArcGIS 9.2)
    • Excel tool to estimate Extraterrestrial Radiation
